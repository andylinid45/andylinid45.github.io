Don't
```
use <font> for text style
```
Do
```
use css
```

Don't
```
use <b> and <i> tags 
```
Do
```
use <strong> tag in stead of <b>, <em> tag in stead of <i>, if you want text to be emphasized
```

Don't
```
use <table> for layout

```
Do
```
use div and css
```

Don't
```
user <br> just for white space
```
Do
```
use css
```
